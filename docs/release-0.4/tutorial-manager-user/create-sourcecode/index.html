소스코드 추가하기

Project 에서 실행하기 위한 소스코드를 추가합니다.

1. 소스코드 관리 페이지 이동​

  • 좌측 메뉴에서 소스코드 클릭

2. 새 프로젝트​

  • 소스코드 화면에서 ⊕ 소스코드 버튼 클릭

3. 소스코드 정보 입력​

  • 소스코드 추가란에서 추가할 소스코드 정보 입력

1. 소스코드 명​

  • 해당 소스코드 이름을 간략하게 정의

1. 소스코드 설명​

  • 해당 소스코드의 설명을 간략하게 입력

2. 소스코드 가져오는 방법​

  • 소스코드를 받을 저장소 정보

    • 파일 업로드

      로컬에서 zip 압축파일을 업로드 하여 소스코드 추가

    • GitHub Public 저장소

      GitHub 에서 Public 으로 저장된 소스코드 클론

    • GitHub Private 저장소

      GitHub Private Repository 에서 소스코드 클론

      • 개인 엑세스 토큰 : github 에서 사용하는 개인 토큰
        • 개인 엑세스 토큰 발급 방법
          1. https://github.com/settings/tokens 접속
          2. New personal access token (classic) 클릭
          3. name 입력
          4. repo 항목 선택후 Generate token 클릭
          5. 생성된 토큰을 추가

4. 소스코드 생성확인​

  • 생성 버튼 클릭시 해당 소스코드가 등록이 됩니다.
    • ◼︎ 해당 소스코드 클릭시 생성한 소스코드의 파일 정보를 확인 가능합니다.
