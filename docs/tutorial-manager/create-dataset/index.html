데이터셋 생성하기

프로젝트 실행간 연동할 데이터셋을 생성합니다.

1. 데이터셋 페이지 이동​

  • 좌측 메뉴에서 데이터셋 페이지 클릭

2. 데이터셋 추가​

  • 데이터셋 화면에서 ⊕ 데이터셋 버튼 클릭

3. 데이터셋 정보 입력​

  • 데이터셋 생성란에서 추가할 데이터셋 정보 입력

  1. 데이터셋 이름​

  2. 데이터셋 설명문​

  3. 데이터셋 추가할 방식 선택​

    1. 파일업로드 : zip 파일을 로컬에서 업로드 하여 추가
    2. GitHub Public 저장소 : GitHub Public Repository 에서 clone
    3. GitHub Private 저장소 : GitHub Private Repository 에서 clone
      • 개인 엑세스 토큰 : github 에서 사용하는 개인 토큰
      • 개인 엑세스 토큰 발급 방법
        1. https://github.com/settings/tokens 접속
        2. New personal access token (classic) 클릭
        3. name 입력
        4. repo 항목 선택후 Generate token 클릭
        5. 생성된 토큰을 추가

4. 데이터셋 생생 확인​

  • 생성 버튼 클릭시 해당 데이터셋이 등록이 됩니다.
    • ◼︎ 해당 데이터셋 클릭시 생성한 데이터셋의 파일 정보를 확인 가능합니다.
