이미지 추가하기

Project 을 생성하기 위한 이미지를 추가합니다.

1. 이미지 관리 페이지 이동​

  • 좌측 메뉴에서 이미지 클릭

2. 새 프로젝트​

  • 이미지 화면에서 ⊕ 이미지 버튼 클릭

3. 이미지 정보 입력​

  • 이미지 추가란에서 추가할 이미지 정보 입력

1. 이미지 설명​

  • 해당 이미지 정보를 간략하게 입력

2. 이미지 저장소​

  • 서버

    ✓ 해당 기능은 Uyuni kubernetes cluster 노드들 내에 이미지가 있어야 합니다.

    서버에서 저장된 이미지 정보 입력

    • 이미지 명 : 서버에서 저장된 이미지 이름
    • 이미지 태그 : 서버에서 저장된 이미지 태그 명

3. project 에서 사용할 외부 port​

  • 실제로 사용할 포트 정보 입력

4. 라이브러리 정보​

  • 화면에 표시할 라이브러리 정보 입력
  • 1. 이미지 관리 페이지 이동
  • 2. 새 프로젝트
  • 3. 이미지 정보 입력
    • 1. 이미지 설명
    • 2. 이미지 저장소
    • 3. project 에서 사용할 외부 port
    • 4. 라이브러리 정보
