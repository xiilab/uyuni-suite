소스코드 추가

워크스페이스 ▷ 소스코드 ▶️ ⨁소스코드 버튼 클릭

소스코드는 3가지 방법으로 추가할 수 있습니다.

1. 파일 업로드​

  • zip 파일 업로드 가능(최대 3G)

2. GitHub Public 저장소​

  • 원격 저장소(Repository)의 URL 입력

3. GitHub Private 저장소​

  1. 원격 저장소(Repository)의 URL 입력
  2. GitHub 개인 액세스 토큰 입력
개인 액세스 토큰 발급 방법

a. https://github.com/settings/tokens 접속

b. New personal access token (classic) 클릭

c. name 입력

d.repo 항목 선택 후 Generate token 클릭

e. 생성된 토큰을 추가
