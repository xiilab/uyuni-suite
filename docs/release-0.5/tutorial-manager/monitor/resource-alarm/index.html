리소스 알람

1. 리소스 알람 추가​

모니터링 ▷ 리소스 알람 ▷ 상단 알림 설정 버튼 클릭 ▶️ ⨁알림 추가 버튼 클릭

  1. 알람 이름 입력
  2. 노드 선택 (복수 선택 가능)

  1. 리소스 알림 조건 추가 (복수 추가 가능)

1. 항목 : GPU, GPU Memory, GPU 온도, Memory, CPU, Disk
2. 연산자 : >, ≥, <, ≤, ==
3. 임계값 : 기준 값 입력
4. 지속 시간 : 입력한 지속 시간 동안 임계값을 지속적으로 초과하거나 유지하면, 자동으로 알림을 전송

  1. 알림 받는 방법 선택

    a. 이메일

    b. Slack

    Slack으로 알림 받기 위해 Slack에서 Webhook 생성 후 입력 필수

    Webhook 생성 방법

2. 이벤트 리스트 조회​

모니터링 ▷ 리소스 알람 ▶️ 상단 이벤트 리스트 버튼 클릭

리소스 알람은 우유니에서도 확인 가능합니다.
