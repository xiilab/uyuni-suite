Kubernetes 프로젝트 생성

프로젝트 ▷ 상단 쿠버네티스 탭 버튼 클릭 ▶️ ⨁프로젝트 버튼 클릭

1. 프로젝트 정보 입력​

1-1 single/Multi Node 선택​

  • Single Node
    • 프로젝트가 단일 노드에서 실행
  • Multi Node
    • 프로젝트가 다중 노드에서 실행
    • horovod를 이용한 분산학습 실행 시 선택

1-2 프로젝트명 입력​

2. GPU 예약하기​

✓ 가용 GPU가 없는 경우 프로젝트를 생성할 수 없습니다.

2-1 GPU 자동 할당 받기​

  1. 기간 입력
  2. GPU 개수 입력
  3. 우유니가 가용 GPU를 자동으로 할당해줍니다.
  4. 할당 받은 GPU 확인

2-2 GPU 직접 선택하기​

  1. 타임테이블에서 원하는 날짜와 시간 선택

    진한 파란색은 이미 예약된 시간 의미

  2. ✓ 와 X 버튼을 클릭하여 GPU 선택 가능

3. 메모리 제한 설정​

프로젝트 생성 ▷ GPU 예약하기 ▶️ 컨테이너 메모리 제한 설정 selectbox에서 설정 가능

✓ 기본적으로 메모리 제한은 'medium (2048m, 4Gi, 4Gi)'로 설정됩니다.

4. 컨테이너 이미지 선택​

원하는 이미지 선택

5. 데이터셋 선택 (옵션)​

  • 원하는 데이터셋 선택 (여러 개 선택 가능)
  • 마운트 경로
    • pod 내부의 컨테이너에서 사용할 수 있는 특정 경로입니다.(경로 변경 가능)
    • 컨테이너 내에서 해당 경로로 접근하여 데이터셋을 불러옵니다.
    • default 경로 /root/uyuni/[유저네임]/dataset/[데이터셋 네임]

6. 소스코드 선택 (옵션)​

  • 원하는 소스코드 선택
  • 마운트 경로
    • pod 내부의 컨테이너에서 사용할 수 있는 특정 경로입니다.(경로 변경 가능)
    • 컨테이너 내에서 해당 경로로 접근하여 소스코드를 실행합니다.

7. 명령어 실행 (옵션)​

  1. 실행 명령어 입력
  • 프로젝트가 생성될 때 실행되는 명령어 입력

    ex) python <file_name> --key1=value1 --key2=value2

  1. 기본 마운트 위치
  • 프로젝트에서 생성된 결과물이 저장되는 디렉토리
