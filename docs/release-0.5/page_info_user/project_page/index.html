프로젝트

생성한 프로젝트 현황을 보여줍니다.

쿠버네티스 프로젝트 페이지​

1. 쿠버네티스 프로젝트 리스트​

쿠버네티스 환경에 생성된 프로젝트 리스트를 출력합니다.

2. 슬럼 프로젝트 리스트​

슬럼 환경에 생성된 프로젝트 리스트를 출력합니다.

3 작업​

해당 프로젝트의 작업을 지정할수 있습니다.

  1. 리소스 모니터링 : 해당 프로젝트의 리소스 정보를 출력합니다.
  2. 웹 터미널 : 해당 프로젝트의 터미널을 출력합니다.
  3. 로그보기 : 해당 프로젝트의 로그를 보여줍니다.
  4. 생성결과 다운로드 : 해당 프로젝트의 파일을 출력하고 이를 다운로드 합니다.
  5. 종료 : 프로젝트를 종료합니다.
  6. 삭제 : 프로젝트를 삭제합니다.
