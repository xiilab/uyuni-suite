Uyuni 의 컨테이너 이미지를 관리하는 페이지 입니다.

1. 전체 페이지​

1.설정​

  • 보고싶은 항목과 기간을 설정 후 적용 버튼을 클릭하면 보고서가 출력됩니다.

2.엑셀, csv 파일 다운​

  • 출력된 보고서를 엑셀 또는 csv 파일로 다운 받을 수 있습니다.

3.리소스 출력​

  • 적용 버튼을 클릭하면 보고서가 출력되는 화면입니다.
